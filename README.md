# Agentic AI Systems with LangChain & LangGraph  
A hands-on repository documenting my journey in building modern agentic AI systems using LangChain, LangGraph, and production-grade LLM engineering techniques.

This repo contains structured code, experiments, and notes as I learn to design AI agents, retrieval systems, multi-agent workflows, and tool-using LLM applications.  
It reflects my practical understanding of how real-world AI systems are built today.

---

## 🚀 What This Repository Covers

### **1. LangChain Foundations**
- Prompt templates  
- Chat models  
- Output parsers  
- LCEL (LangChain Expression Language)  
- Debugging & tracing with LangSmith  
- Using local open-weight models (Ollama)

### **2. Agentic AI Concepts**
- What agents are and how they reason  
- Tool calling & function calling  
- ReAct prompting  
- Multi-agent collaboration  
- Context engineering & memory

### **3. Retrieval-Augmented Generation (RAG)**
- Embeddings  
- Vector stores (FAISS, Pinecone, Chroma)  
- Retrieval pipelines  
- Document loaders  
- Building a documentation assistant

### **4. LangGraph Workflows**
- State graphs  
- Branching logic  
- Multi-agent orchestration  
- Reflection & self-correction loops  
- Agentic RAG with LangGraph

### **5. Model Context Protocol (MCP)**
- Introduction to MCP  
- Using pre-built MCP servers  
- Building custom MCP servers & clients

### **6. Production-Ready LLM Engineering**
- Agent security  
- Common vulnerabilities in AI apps  
- Best practices for LLM systems  
- Observability with LangSmith
